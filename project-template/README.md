Project Template

Structure:

- data/raw/ # original data
- data/processed/ # cleaned/derived data
- notebooks/ # exploratory notebooks
- src/ # reusable scripts and modules
- models/ # trained model artifacts
- reports/ # plots, PDFs, notebooks exported

Usage:

1. Copy this folder to start a new project.
2. Fill `requirements.txt` or `environment.yml`.
3. Document the project in `README.md`.
